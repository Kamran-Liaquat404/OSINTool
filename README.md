# OSINTool v2.0 – Complete OSINT Recon & Investigation Suite

**OSINTool v2.0** is a powerful all-in-one OSINT (Open Source Intelligence) tool designed for cybersecurity enthusiasts, bug bounty hunters, and digital investigators. It provides domain, IP, email, and username analysis with automated correlation, risk scoring, and multi-format report generation.

---

## 🌐 Features

**Domain Analysis:** IP resolution, DNS records (A, MX, TXT, NS, CNAME, AAAA, SOA), subdomain finder (CRT.sh), DNS bruteforce, SSL checks, Wayback snapshots, WHOIS lookup, tech stack detection.  

**IP Analysis:** Geolocation, ISP info, reverse DNS, open port scanning, reverse IP lookup.  

**Email Analysis:** Validation, MX check, Gravatar detection, disposable email detection, email pattern analysis, related username generator.  

**Username & Social Analysis:** 55+ platform check, multi-threaded scanning, auto-save results.  

**Google Dork Generator:** Sensitive files, login pages, exposed info, subdomains, cached pages.  

**Auto Correlation Engine:** Domain → IP → Subdomains → Emails, Email → Username → Platforms, IP → Location → ISP → Domains.  

**Risk Score System:** 0–100 score, LOW/MEDIUM/HIGH/CRITICAL, detailed breakdown.  

**Reports:** JSON, HTML, PDF, TXT auto-save for professional use.  

**Technical:** Termux & Android compatible, multi-threaded, no API keys required, auto target detection.
